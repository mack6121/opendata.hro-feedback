# OpenData.HRO-Feedback


## Background

**OpenData.HRO-Feedback** is a web application for feedback administration closely related and connected to OpenData.HRO. It is based on [Django](https://github.com/django/django).

The project was initiated by municipality of Rostock (Germany), more precisely its land registry office.


## In production

**OpenData.HRO-Feedback** is running in production on the project initiators' servers: http://www.opendata-hro.de/feedback/en/requested-datasets
